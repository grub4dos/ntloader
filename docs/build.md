## Compilation

gcc, gcc-mingw-w64 and gcc-aarch64-linux-gnu are required to compile NTloader.  

### Compile NTloader
```
# Build everything
make
# Build arm64 binary only
make ntloader.arm64
```

### Compile fsuuid
```
make fsuuid
make fsuuid.exe
```

### Compile bmtool
```
make bmtool
make bmtool.exe
```

### Compile mkinitrd
```
make mkinitrd
make mkinitrd.exe
```

### Build initrd.cpio
```
make initrd.cpio
```
